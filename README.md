# Cybersecurity-Projects
------------------------------------------------------------------------------------------------------
1. Advanced Port Scanner
A multi-threaded port scanner designed to identify open ports and detect running services on a target system.

Features:
	•	Multi-threaded scanning for faster performance
	•	Detects open ports (1–1024)
	•	Banner grabbing for service identification
	•	Lightweight and efficient

Tech Stack: Python (socket, threading)

Use Case:
Helps security professionals identify exposed services and potential attack vectors.

------------------------------------------------------------------------------------------------------
2. Web Vulnerability Scanner

A basic web security scanner to detect common vulnerabilities in web applications.

Features:
	•	Detects SQL Injection (basic payload testing)
	•	Detects Cross-Site Scripting (XSS)
	•	Checks for missing security headers
	•	Simple and fast scanning

Tech Stack: Python (requests)

Use Case:
Useful for identifying common web vulnerabilities during initial security assessments.

------------------------------------------------------------------------------------------------------
3. Log Analyzer (SOC Tool)
   
A log analysis tool designed to detect suspicious activity and brute-force attempts.

Features:
	•	Detects failed login attempts
	•	Identifies suspicious IP addresses
	•	Highlights brute-force attack patterns
	•	Simple SOC-style alerting

Tech Stack: Python (regex)

Use Case:
Helps security analysts monitor logs and detect potential intrusions.
